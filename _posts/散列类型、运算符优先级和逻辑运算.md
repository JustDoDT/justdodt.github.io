### 1. 集合

#### 1.1 集合的2种定义方法


#### 1.2 集合中的交集并集和差集

~~~python

In [65]: a - c
Out[65]: set()
~~~


##### 1.3.2 集合的删除

~~~python
In [3]: a
Out[3]: {1, 1234, 2, 3, 34, 4, 99, 'a', 'b', 'c', 'hhaha', 'python', 'w'}

In [4]: a.pop()
Out[4]: 1         #pop()是随机删除，不是和列表一样删除末尾的元素，因为集合无序性。不支持索引

In [5]: a.pop()
Out[5]: 2

In [7]: a
Out[7]: {1234, 34, 4, 99, 'a', 'b', 'c', 'hhaha', 'python', 'w'}
In [8]: a.remove('hhaha')  #remove()是指定元素删除
In [9]: a
Out[9]: {1234, 34, 4, 99, 'a', 'b', 'c', 'python', 'w'}
    
~~~

##### 1.3.3 集合的查询

~~~python
In [11]: a = {1,2,3,'ha'}
In [12]: b = {1,2}
In [14]: c = {6,7,8}

In [17]: a.isdisjoint(c) 
Out[17]: True  		#isdisjoint()是判断2个集合是否有交集，如果没有交集，则返回Ture,反之成立
In [18]: a.isdisjoint(b)
Out[18]: False

In [13]: a.issubset(b) #判断集合a是否是集合b的子集？
Out[13]: False
In [15]: a.issubset(c)
Out[15]: False   #判断集合a是否包含于集合c，即集合的子集问题
In [19]: b.issubset(a)
Out[19]: True    #判断集合b是否包含于集合a，即集合b是否是集合a的子集。
    
In [20]: b.issuperset(a)
Out[20]: False  #判断集合的子集问题，判断集合b是否被包含于集合a
In [21]: a.issuperset(b) #判断集合a被包含于集合b，即集合b是否是集合a的子集？
Out[21]: True
    
~~~

### 2. 字典

#### 2.1 字典的2种定义方法

~~~python
#第一种方法：通过键值的形式定义，在字典里键值是唯一的，但是值可以不唯一
In [28]: a = {'name':'zhangsna','age':18,'sex':'男'}
In [29]: a
Out[29]: {'name': 'zhangsna', 'age': 18, 'sex': '男'}
In [30]: a['name']   
Out[30]: 'zhangsna'
In [31]: a['age']
Out[31]: 18

#第二种方法：通过函数的形式定义   
In [36]: b = dict(name='zhangsan',age=18)
In [37]: b
Out[37]: {'name': 'zhangsan', 'age': 18}
In [38]: b['name']   #通过键获取值
Out[38]: 'zhangsan'

~~~

#### 2.3 字典的总结

- 唯一性：字典的键（key）是唯一的，不存在2个相同的键(key)
- 可变性：字典是可变对象，但是字典的键（key）必须是不可变对象
- 无序性：字典中的键也是无序的，所以不能通过索引取值

### 4. 总结数值类型，序列类型，散列类型

数值类型（整型，浮点型，复数，布尔型（0，1）），

序列类型（列表，字符串，元组），

散列类型（集合，字典）；

其中不可变的类型有：数值类型，元组，字符串类型；

可变类型有列表，集合，字典。

~~~python
# 数值类型
a = 1      #整型
a = 1.2    #浮点型
a = 1 + 2j #复数类型
st = 'abc' #bool类型
'b' in st 

#序列类型
a = [1,2,3]   #列表
b = 'hello world'  #字符串
c = (1,2,3,4)	   #元组

#散列类型
a = {1,2,3}		#集合
a = [1,2,3,1]
b = set(a)		#集合
b = {'name':'lisi'}   #字典
b = dict(name='zhangsan',age = 18) #字典
~~~
